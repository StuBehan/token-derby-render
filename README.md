# Token Derby Render

Vue + Three.js prototype for rendering a stylized horse race with track rails, fences, a stand, animated runners, and a finish line.

## Setup

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` starts the Vite development server.
- `npm run build` type-checks and builds the app.
- `npm run preview` serves the production build locally.
